A WooCommerce plugin for Israeli Orange Delivery.

== Description ==
A WooCommerce plugin for Israeli Orange Delivery.
Note! you must have a customer code to use this plugin!

== FEATURES LIST ==
* Control your order with a new status "delivery".
* Every order in "delivery" status get a delivery number from Tapuz.
* Print Israeli Orange Delivery Shipping Label directly on a sticker.
* Change order status to "completed" automatically once delivery is done.
* Easy to use.
* Compatible to all WooCommerce Themes.

We customize this plugin with new functionality, adapted to every client.
[Contact us](https://gamliel.solutions/contact-gamliel-solutions/) for more details

== Installation ==
* plugins -> Add New.
* search for 'Israeli Orange Delivery'.
* click on Install Now and then Active.

== Using FTP or similar ==
* unzip 'israeli-orange-delivery' file and 
* Upload 'israeli-orange-delivery' folder to the /wp-content/plugins/' directory.
* Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Where can I find plugin settings? =

Login to your admin account, click on the settings under the WooCommerce menu, click on the Orange Delivery tab.
